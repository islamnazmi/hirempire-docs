## Query Parameters

<ParamField query="job_id" type="string" required placeholder="Enter job ID">
  The unique identifier of the job to retrieve applicants for
</ParamField>

## Authentication

<ResponseField name="Authorization" type="string" required>
Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token. 
</ResponseField>

```rust Rust
use reqwest::header::{HeaderMap, HeaderValue, AUTHORIZATION};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut headers = HeaderMap::new();
    headers.insert(AUTHORIZATION, HeaderValue::from_str("Bearer YOUR_TOKEN")?);
    
    let client = reqwest::Client::new();
    let response = client
        .get("https://api.hirempire.com/v1/applicant")
        .query(&[("job_id", "{job_id}")])
        .headers(headers)
        .send()
        .await?;
        
    let body = response.text().await?;
    println!("{}", body);
    
    Ok(())
}
```

## Response

<ResponseField name="success" type="boolean">
Indicates if the request was successful
</ResponseField>

<ResponseField name="total_applicants" type="integer">
Total number of applicants for this specific job
</ResponseField>

<ResponseField name="applicants" type="array">
Array of applicant objects for the specified job

<Expandable title="Applicant Object Properties">
<ResponseField name="id" type="string">
Unique identifier for the applicant
</ResponseField>

<ResponseField name="applied_at" type="string">
ISO 8601 timestamp when the application was submitted
</ResponseField>

<ResponseField name="name" type="string">
Full name of the applicant
</ResponseField>

<ResponseField name="job_title" type="string">
Title of the job the applicant applied for
</ResponseField>

<ResponseField name="company_name" type="string">
Name of the company for the job application
</ResponseField>

<ResponseField name="first_name" type="string">
First name of the applicant
</ResponseField>

<ResponseField name="email" type="string">
Email address of the applicant
</ResponseField>

<ResponseField name="phone_number" type="string">
Phone number of the applicant
</ResponseField>

<ResponseField name="linkedin_url" type="string">
LinkedIn profile URL of the applicant
</ResponseField>

<ResponseField name="source" type="string">
Platform or source where the application came from
</ResponseField>

<ResponseField name="cv_url" type="string">
URL to download the applicant's CV/resume
</ResponseField>

<ResponseField name="status" type="string">
Current status of the application (e.g., Applied, Accepted, Rejected)
</ResponseField>

<ResponseField name="score" type="integer">
Scoring or rating assigned to the applicant
</ResponseField>
</Expandable>
</ResponseField>
