---
title: "Update job status"
api: "PATCH https://api.hirempire.com/v1/update-job"
description: "Update the status of an existing job posting in your Hirempire account"
---

## Request Body

<ParamField body="job_id" type="string" required placeholder="Enter job ID">
  The job ID to retrieve
</ParamField>

<ParamField body="status" type="string" required>
The new status for the job. 
Valid values: `Pending`, `Active`, `Paused`, `Hired`, `Closed`
</ParamField>


## Authentication

<ResponseField name="Authorization" type="string" required>
Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token. 
</ResponseField>

```java Java
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;

public class HirempireAPI {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
            "job_id": "xyz456abc123",
            "status": "Active"
        }
        """;
        
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://api.hirempire.com/v1/update-job"))
            .header("Authorization", "Bearer YOUR_TOKEN")
            .header("Content-Type", "application/json")
            .method("PATCH", HttpRequest.BodyPublishers.ofString(payload))
            .build();
            
        HttpResponse<String> response = client.send(request, 
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

```rust Rust
use reqwest::header::{HeaderMap, HeaderValue, AUTHORIZATION, CONTENT_TYPE};
use serde_json::json;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut headers = HeaderMap::new();
    headers.insert(AUTHORIZATION, HeaderValue::from_str("Bearer YOUR_TOKEN")?);
    headers.insert(CONTENT_TYPE, HeaderValue::from_str("application/json")?);
    
    let payload = json!({
        "job_id": "xyz456abc123",
        "status": "Active"
    });
    
    let client = reqwest::Client::new();
    let response = client
        .patch("https://api.hirempire.com/v1/update-job")
        .headers(headers)
        .json(&payload)
        .send()
        .await?;
        
    let body = response.text().await?;
    println!("{}", body);
    
    Ok(())
}
```

## Response

<ResponseField name="success" type="boolean">
Indicates if the job status was updated successfully
</ResponseField>

<ResponseField name="id" type="string">
The unique identifier of the updated job
</ResponseField>

<ResponseField name="status" type="string">
The new status of the job
</ResponseField>

<ResponseExample>
```json

  "success": true,
  "id": "xyz456abc123",
  "status": "Active"
}
```
